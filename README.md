# Full-Stack CRUD Todo App #
Simple full-stack todo app built with:
- Node.js + Express
- File-based persistence (`data.json`)
- REST API (GET, POST, PUT, DELETE under `/api/items`)
- Vanilla JavaScript frontend served from Express (`/public/index.html`)
## Run locally ##
npm install,

node server.js
Then open:
- API: http://localhost:3000/api/items  
- Frontend: http://localhost:3000/index.html
